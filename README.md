# Titanic Survival Prediction

A machine learning project that predicts passenger survival on the Titanic using historical data. Built with Python, featuring a Random Forest classifier trained on the Kaggle Titanic dataset.

## Features
- Interactive Streamlit web app for real-time predictions
- Data preprocessing and model training script
- Feature importance analysis and survival statistics visualizations
- User-friendly interface for inputting passenger details

## Tech Stack
- Python
- Scikit-learn (Random Forest)
- Streamlit
- Pandas, NumPy, Matplotlib, Seaborn

## Usage
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
4. Or train the model: `python titanic_model.py`

## Deployment on Streamlit Cloud
1. Push your code to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub account
4. Select this repository
5. Set the main file path to `app.py`
6. Click Deploy

Dataset: [Kaggle Titanic Dataset](https://www.kaggle.com/c/titanic/data)
